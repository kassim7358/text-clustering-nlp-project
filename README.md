# Machine Learning Classification Project

## Objective
The aim of this project is to build and evaluate machine learning models for a classification problem.

## Methods
- Data preprocessing and cleaning
- Feature engineering
- Model training using:
  - Logistic Regression
  - Decision Tree
  - Random Forest (if used)
- Model evaluation using:
  - AUROC
  - Precision
  - Recall
  - F1-score

## Results
Models were compared based on performance metrics, highlighting trade-offs between precision and recall.

## Key Insights
- Different models perform differently depending on the dataset
- Evaluation metrics are essential for understanding model performance beyond accuracy
- Feature engineering significantly impacts results

## Tools Used
Python, pandas, scikit-learn, matplotlib

## Note
This project was developed as part of MSc coursework and refined for portfolio purposes.
