# Text Clustering using NLP Techniques

## Objective
To cluster textual data using TF-IDF vectorisation and KMeans clustering.

## Methods
- Text preprocessing and cleaning
- TF-IDF feature extraction
- KMeans clustering
- Evaluation of cluster structure

## Key Insights
- Text data can be transformed into numerical features using TF-IDF
- Clustering reveals hidden structure in unlabelled data

## Tools Used
Python, pandas, scikit-learn
